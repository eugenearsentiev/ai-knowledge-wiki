# DALL·E

[[openai|OpenAI]]'s text-to-image model series. DALL·E (2021) demonstrated text-conditioned image generation; DALL·E 2 (2022) improved quality and editability; DALL·E 3 (2023) substantially improved prompt adherence.

**Architecture** — uses [[diffusion-models|diffusion]] with CLIP-based text conditioning. DALL·E 3 was trained on images recaptioned by GPT-4 for better instruction following.

**Access** — closed/API only, with no publicly released weights. Available via OpenAI API and integrated into [[chatgpt|ChatGPT]] for image generation.

**Competition** — Midjourney (known for aesthetics), [[stable-diffusion|Stable Diffusion]] (open-weight, local), Adobe Firefly (commercially safe, enterprise-focused).

**Model vs. product** — DALL·E is the model; image generation within ChatGPT is the product interface.

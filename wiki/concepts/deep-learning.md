# Deep Learning

The branch of [[machine-learning|machine learning]] that uses neural networks with many layers. More layers allow the network to learn increasingly abstract representations — edges → shapes → objects, for instance.

**Why it worked** — the combination of large datasets, GPU compute, and architectural innovations (especially the [[transformer|Transformer]]) drove a step-change in capability starting with [[alexnet-imagenet|AlexNet (2012)]].

**Core mechanics** — feedforward networks ([[mlp|MLP]]), backpropagation, gradient descent, activation functions (ReLU), regularization (dropout).

**Main architectural families** — [[cnn|CNNs]] (spatial/image data), [[rnn-lstm|RNNs/LSTMs]] (sequences, pre-Transformer), [[transformer|Transformers]] (sequences, images, multimodal — the current dominant family), [[diffusion-models|diffusion models]] (generation), [[gan|GANs]] (generation, now largely displaced).

**Key figures** — [[geoffrey-hinton|Hinton]], [[yann-lecun|LeCun]], [[yoshua-bengio|Bengio]] — the "Godfathers of Deep Learning," 2018 Turing Award.

**Hierarchy** — [[machine-learning|Machine Learning]] → Deep Learning → [[generative-ai|Generative AI]].

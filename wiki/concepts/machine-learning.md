# Machine Learning

The branch of AI where systems improve at a task by learning patterns from data, rather than following rules written by hand. Almost everything in modern AI rests on this idea.

**The three paradigms:**
- **Supervised learning** — labeled input-output pairs; learn to predict outputs for new inputs. Tasks: regression (predict a number), classification (predict a category).
- **Unsupervised learning** — unlabeled data; find structure. Tasks: clustering, dimensionality reduction.
- **[[reinforcement-learning|Reinforcement learning]]** — agent learns from environment feedback (rewards/penalties).

**Core concepts** — features, training/test split, overfitting vs. underfitting (bias-variance tradeoff), evaluation metrics.

**Classical algorithms** — see [[classical-ml|Classical ML]] (stub; detailed treatment in a separate focused KB).

**Where neural networks fit** — neural networks are one ML model family, especially powerful for unstructured data; "deep learning" = neural nets with many layers. See [[deep-learning|Deep Learning]].

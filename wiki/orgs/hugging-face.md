# Hugging Face

The field's shared open hub for models, datasets, and ML infrastructure. Founded 2016, originally as a chatbot company, pivoted to become the de facto repository for open ML.

**Core offerings** — the `transformers` library (Python; standard way to load and use [[transformer|Transformer]] models), `datasets` library, the Hub (host/share/discover models and datasets), Spaces (deploy demos and apps), `peft` (parameter-efficient fine-tuning including [[lora|LoRA]]), `diffusers` ([[diffusion-models|diffusion models]]).

**Role in the ecosystem** — if an open-weight model exists, it's almost certainly on Hugging Face. The Hub hosts [[llama|Llama]], [[bert|BERT]], [[stable-diffusion|Stable Diffusion]], and tens of thousands of fine-tuned variants.

**Enterprise** — Hugging Face also provides hosted inference endpoints and enterprise-grade features.

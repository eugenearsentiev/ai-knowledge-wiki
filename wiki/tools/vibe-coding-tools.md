# Vibe Coding Tools

**Vibe coding tools** are AI-assisted software builders where the user describes intent in natural language and the tool writes, edits, previews, or deploys code. They sit at the intersection of [[generative-ai|Generative AI]], [[llm|LLMs]], [[prompting]], and [[agents-tool-use|agentic tool use]].

Two broad clusters matter. App builders such as Lovable, Replit Agent, Bolt, v0, Base44, and Firebase Studio focus on turning ideas into working prototypes or small apps, often with hosting, previews, and visual editing built in. Developer tools such as Cursor, GitHub Copilot, Claude Code, Codex, Devin Desktop/Windsurf, and Devin focus on existing repositories, multi-file edits, tests, pull requests, and code review.

The practical tradeoff is speed versus control. These tools are excellent for scaffolding, UI iteration, internal tools, and exploratory builds, but production use still needs review for architecture, security, data handling, and maintainability.

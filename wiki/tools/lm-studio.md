# LM Studio

A desktop application (Mac, Windows, Linux) for discovering, downloading, and running open-weight [[llm|LLMs]] locally. Provides a graphical UI for model management and chat, and a local server with an OpenAI-compatible API.

**Compared to [[ollama|Ollama]]** — Ollama is CLI-first and developer-focused; LM Studio is GUI-first and more accessible for less technical users. Both use [[quantization|quantized]] models (GGUF format) and llama.cpp under the hood for inference.

**Use cases** — same as Ollama: private/local deployment, offline use, cost-free experimentation. Particularly popular with users who prefer a visual interface over the command line.

**Model discovery** — integrates with [[hugging-face|Hugging Face]] model search for browsing and downloading models directly from the app.
